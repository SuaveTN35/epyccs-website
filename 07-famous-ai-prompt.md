## ALTERNATIVE: SQUARESPACE IMPLEMENTATION GUIDE

If rebuilding on Squarespace instead of Famous.ai:

### Recommended Template:
- Brine family (most flexible)
- Bedford (professional services)
- Aviator (modern, clean)

### Page Structure:
1. Create pages: Home, Medical Courier, Legal Courier, Commercial Delivery, About, Contact
2. Set up navigation with Services as dropdown
3. Add announcement bar with phone number

### Key Squarespace Features to Use:
- Form blocks for quote requests
- Gallery blocks for service areas
- Summary blocks for blog posts
- Button blocks for CTAs
- Spacer blocks for layout
- Code injection for schema markup

### Squarespace SEO Settings:
1. Go to each page → Settings → SEO
2. Add title tag and meta description from SEO package
3. Add social sharing image
4. Enable clean URLs

### Code Injection (Settings → Advanced → Code Injection):
Add schema markup in header injection area.
